## 👩🏻‍💻 프로그램 실행하는 법
```
1️⃣ 레포지토리 클론
git clone https://github.com/1mptera/web-03-project01-bohyunkang.git

2️⃣ 디렉터리 이동
cd web-03-project01-bohyunkang

3️⃣ 프로젝트 소스 코드 컴파일
./gradlew compileJava
java -cp build/classes/java/main ClassName

4️⃣ 프로그램 실행
./gradlew run

5️⃣ 테스트 코드 실행
./gradlew test
```
